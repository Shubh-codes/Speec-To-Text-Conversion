# Speech-To-Text Conversion

## Project Overview

Speech-To-Text Conversion is a Python-based application that converts spoken audio into text using speech recognition technology. The application listens to the user's voice through a microphone, processes the audio input, and transcribes it into text using Google's Speech Recognition API.

The project also includes text-to-speech functionality to provide voice prompts and interact with the user, creating a simple voice-enabled interface.

## Tech Stack

### Programming Language

* Python

### Speech Recognition

* SpeechRecognition
* Google Speech Recognition API

### Text-to-Speech

* pyttsx3

### Other Libraries

* datetime
* audioop

## Key Features

* Captures voice input through the system microphone.
* Converts speech into text in real time.
* Supports English (India) speech recognition.
* Provides voice-based greetings and instructions.
* Uses text-to-speech to interact with the user.
* Simple and lightweight implementation for learning speech processing concepts.
